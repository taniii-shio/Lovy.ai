# Lovy.ai - プロジェクト概要

## 1. プロジェクト名

**Lovy.ai** - MBTI × ラブタイプ恋愛性格診断アプリケーション

## 2. 目的

MBTI（16 タイプ）とラブタイプ診断（16 タイプ）を掛け合わせた 256 通りの恋愛性格診断を提供し、ユーザーの恋愛における特性を多角的に分析・可視化する。

## 3. ターゲットユーザー

- 恋愛における自己理解を深めたい人
- パートナーとの相性を知りたい人
- SNS でシェアできる診断コンテンツを求める人

## 4. コアバリュー

- **256 通りの組み合わせ**: MBTI（4 文字）× ラブタイプ（4 文字）= 詳細な診断
- **5 つの独自診断アルゴリズム**: 科学的根拠に基づいた多角的分析
- **SNS シェアビリティ**: 結果を簡単に共有できる UX

## 5. 提供する診断項目

1. **魅力度診断**: 数値化された総合的な魅力スコア
2. **性格軸分析**: 上位 3 つの性格特性とポジティブな分析
3. **相性ベスト 3**: 256 タイプの中から最も相性の良い組み合わせ
4. **愛の言語優先順位**: 5 つの愛情表現スタイルのランキング
5. **人生配分**: 理想的なライフバランスの可視化

## 6. 主要機能

### Must Have（必須機能）
- 診断実行（MBTI × Love Type 16 選択 → 結果出力）
- 結果表示（5つの診断項目）
- **結果サマリ画像のダウンロード（PNG形式）**

### Nice to Have（初期リリースで可能なら）
- SNS 共有機能（Twitter, Instagram, LINE 等）
